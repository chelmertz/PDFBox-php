#PDFBox-php
Parsing text from PDF-files using the java utility [PDFBox](http://pdfbox.apache.org/) Apache.

##Requirements
 - java
 - [PDFBox](http://pdfbox.apache.org/) - one single .jar file necessary - and it must be executable
 - PHP 5.3 (for namespaces)
